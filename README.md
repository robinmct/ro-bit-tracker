# Ro-bit

A modern habit tracker built with Next.js, React, shadcn/ui, and Firebase.

## Tech Stack
- **Framework**: Next.js 16 + React 19
- **Styling**: Tailwind CSS + shadcn/ui
- **State**: Zustand
- **Backend**: Firebase Auth + Firestore
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Push to main branch on GitHub and connect to Vercel.
